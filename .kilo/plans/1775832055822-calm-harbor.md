# Plan: Design System Update - Cinematic AI SaaS Theme

## Overview
Update the design system across all apps (excluding Render page, timeline editor, AI storyboarder, and templates app) with a new cinematic AI SaaS color palette that combines video editing tools, AI SaaS aesthetics, glassmorphism, and dark cinematic environments. This creates a cohesive brand identity while maintaining functional differences across apps.

## Current State Analysis
- **Current Theme**: Dark mode with neon yellow-green (#d9ff00) and purple accents
- **Excluded Apps**: RenderPage, TimelineEditorPage, StoryboardStudio, AIStoryboardPage, TemplatesPage, TemplateStudio, CinemaTemplateStudio, CinematicTemplateWizard
- **Apps to Update**: All other 35+ components including studios, pages, and utilities
- **CSS Structure**: Variables in `variables.css`, Tailwind theme in `global.css`, component-specific styles

## New Color System Specification

### 🎨 Core Foundation
**Background Layer:**
- Primary: `#0a0f19` → `#060910` (deep navy to near black)
- Creates cinematic feel, high contrast for media, "editing studio" vibe

**Atmospheric Glow:**
- Blue glow: `rgba(78,126,255,0.18)`
- Cyan glow: `rgba(67,212,255,0.12)`
- Applied via radial gradients throughout the interface

### 🔵 Brand Identity
**Primary Gradient (Most Important):**
- Start: `#3f66ff` (electric blue)
- End: `#3acfff` (cyan)
- Used on buttons, logo, primary CTAs

### 🧊 Surface & UI Elements
**Cards/Panels:**
- Background: `rgba(11,17,28,0.9)`
- Border: `rgba(255,255,255,0.08)`
- Creates glassmorphism feel with layered depth

**Interactive Elements:**
- Default buttons: `rgba(255,255,255,0.06)` → `rgba(255,255,255,0.10)` on hover
- Primary buttons: `#3f66ff` → `#3acfff` gradient with `rgba(58,126,255,0.35)` glow

### 🎬 Preview & Stage Areas
**Stage Background:**
- Base: `#05070c` → `#0c1018`
- Glow: `rgba(120,170,255,0.10)`

**Preview Screen:**
- Background: `#1b2742` → `#0f1728`

### 🧠 AI & Tool Accents
**Subtle Intelligence Indicators:**
- Blue: `rgba(95,129,255,0.25)`
- Cyan: `rgba(58,207,255,0.16)`

### 📝 Typography
**Text Hierarchy:**
- Primary: `#eef4ff`

## Technical Implementation Details

### CSS Variable Migration Strategy
```css
/* OLD variables.css */
--color-primary: #d9ff00;
--bg-app: #050505;

/* NEW variables.css */
--bg-primary: #0a0f19;
--bg-primary-dark: #060910;
--glow-blue: rgba(78,126,255,0.18);
--glow-cyan: rgba(67,212,255,0.12);
--brand-gradient-start: #3f66ff;
--brand-gradient-end: #3acfff;
--surface-card: rgba(11,17,28,0.9);
--border-surface: rgba(255,255,255,0.08);
--text-primary: #eef4ff;
--text-secondary: #90a3bf;
```

### Component Update Pattern
```javascript
// Before
className="bg-card text-primary border-border"

// After
className="bg-surface-card text-text-primary border-border-surface"
style={{
  background: 'linear-gradient(135deg, var(--bg-primary), var(--bg-primary-dark))',
  boxShadow: 'var(--glow-blue)'
}}
```

### Atmospheric Glow Implementation
```css
.glass-panel {
  background: var(--surface-card);
  border: 1px solid var(--border-surface);
  backdrop-filter: blur(20px);
  position: relative;
}

.glass-panel::before {
  content: '';
  position: absolute;
  inset: 0;
  background: radial-gradient(circle at center, var(--glow-blue), transparent 70%);
  pointer-events: none;
}
```
