> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Integrate ALL features from CineGen, LTX-Desktop, rendiv, timeline-studio, chatvideo-yucut, and remix-go into the existing Higgsfield TimelineEditorPage.js as a **native component (NO IFRAME)**.

- **Animation:** Framer Motion or CSS animations

**Key Change:** Remove iframe isolation, implement as native component for better performance, debugging, and feature integration.

## Repository Analysis: Timeline Studio Features

### Core Data Model (Phase 1)
**Source:** https://github.com/deangilmoreremix/timeline-studio

**Key Features to Integrate:**

**Project Entity:**
```javascript
project = {
  id,
  fps: 30,
  duration: 180000, // ms
  aspectRatio: '16:9',
  tracks: [],
  assets: [],
  markers: [],
  captions: [],
  effects: []
}
```

**Track Entity:**
```javascript
track = {
  id,
  type: 'video' | 'audio' | 'text' | 'captions' | 'effects',
  name,
  locked: false,
  muted: false,
  items: []
}
```

**Item Entity:**
```javascript
item = {
  id,
  assetId,
  type: 'video' | 'audio' | 'image' | 'text',
  start: 0,        // ms
  end: 5000,       // ms
  sourceStart: 0,  // trim in
  sourceEnd: 5000, // trim out
  lane: 0,
  x: 0,            // position in timeline
  width: 100,      // duration in pixels
  trimIn: 0,
  trimOut: 0,
  volume: 1.0,
  playbackRate: 1.0,
  effects: []
}
```

### Multi-Track Support (Phase 1)
**Features:**
- Video Track (primary video clips)
- Audio Track (voiceovers, music, sound effects)
- Caption Track (subtitles, text overlays)
- Effects Track (transitions, filters)
- Overlay/Text Track (titles, lower thirds)

### Timeline Interactions (Phase 2)
**Features:**
- **Clip Operations:** Drag to reposition, resize edges for trim
- **Split at Playhead:** Cut clips at current timeline position
- **Snap System:** Auto-align clips to nearby items/markers
- **Zoom/Pan:** Timeline magnification and horizontal scrolling
- **Selection:** Single/multi-clip selection with visual feedback

**Implementation:** Pointer Events API with shared `editorState` object.

### Waveform Support (Phase 3)
**Features:**
- **Audio Visualization:** Precomputed waveform peaks stored in assets
- **Timeline Integration:** Draw waveforms directly in clip components
- **Performance:** Use wavesurfer.js or Canvas-based rendering
- **Sync Features:** Audio alignment, dead-air detection, caption timing

### Preview-Sync Engine (Phase 3)
**Features:**
- **Multi-Source Sync:** Playhead ↔ video element ↔ waveform ↔ captions
- **Controller Pattern:**
```javascript
const playbackEngine = {
  playing: false,
  currentTime: 0,
  fps: 30,
  seek(time) {},
  play() {},
  pause() {},
  syncPreview() {},
  syncTimeline() {},
  syncCaptions() {}
}
```

### Editor Features (Phase 4)
**UI Components:**
- **Multi-track Timeline:** Expand from single strip to full track system
- **Clip Trim Handles:** Visual resize controls on clip edges
- **Playhead Scrubber:** Precise timeline navigation
- **Timeline Markers:** Scene breaks, chapter points
- **Inspector Panel:** Clip properties, effects, timing
- **Keyboard Shortcuts:** Standard NLE shortcuts (V: select, B: blade, etc.)

### Caption System (Phase 4)
**Features:**
- **Caption Tracks:** Dedicated text overlay lanes
- **Caption Blocks:** Timed text elements on timeline
- **Style Panel:** Font, color, animation presets
- **Drag Retiming:** Move captions along timeline
- **Per-Caption Animation:** Fade in/out, typewriter effects

### AI-Assisted Editing (Phase 5)
**Features:**
- **Timeline Patches:** AI returns structured edit operations
- **Smart Actions:** Cut dead space, find highlights, make shorts, add captions
- **Patch Application:**
```javascript
// AI returns timeline modifications
{
  actions: [
    { type: 'splitClip', clipId: 'c1', time: 12.4 },
    { type: 'removeClipRange', clipId: 'c1', start: 12.4, end: 18.9 },
    { type: 'addCaptionTrack', captions: [...] },
    { type: 'createSequence', trackId: 'video-2', clips: [...] }
  ]
}
```

function handlePointerDown(event, itemId) {

function handlePointerMove(event) {
  if (editorState.dragging) {
    moveItem(editorState.dragging.itemId, event.clientX);

    resizeItem(editorState.resizing.itemId, event.clientX);
  }
}

function drawWaveform(canvas, waveform, width, height) {
  const ctx = canvas.getContext('2d');
  const step = waveform.length / width;

  ctx.beginPath();
  for (let x = 0; x < width; x++) {
    const peak = waveform[Math.floor(x * step)];
    const y = (1 - peak) * height / 2;
    ctx.lineTo(x, y);
  }
  ctx.stroke();
}
